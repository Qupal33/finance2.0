# finance



запуск проекта

git clone https://github.com/Qupal33/finance.git

cd finance

python -m venv venv

venv\\Scripts\\activate

source venv\\Scripts\\activate

pip install -r requirements.txt

python app.py



Приложение будет доступно по адресу:

http://127.0.0.1:5000



После первого запуска, приложение автоматически создаст базу данных и необходимые таблицы. Для этого будет использован SQLite.
